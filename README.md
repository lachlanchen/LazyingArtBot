## 你是否也有這種痛苦

```
☕ 咖啡館                  💼 開完會                   🌙 睡前
─────────────              ──────────────              ──────────────

> 想到了 → 沒記 → 消失

**Kairo 就是為了打破這個循環而生的。**

└─────────────────────────────────────────────────────────────────────┘
```

### Kairo 做到了什麼別人沒做到的事

**1. 它在你常用的 IM 裡 — 零切換成本**

ChatGPT Pulse 再好，你要打開 App 才能看。Notion AI 再強，你要進 Notion 才能用。  
Kairo 在你的 Telegram 或 Feishu 群組裡，你本來就在這裡。

**2. 它主動找你，不是你主動找它**

傳統工具是「拉模型」(Pull)：你想起來才去查。  
Kairo 是「推模型」(Push)：到了時間它就來找你，哪怕你忘了。

```
08:00  → 晨報：今日行程 + 逾期任務 + 昨日郵件摘要
09:00  → 「你說過今天要跟 Jason 確認合同，現在是個好時機」
20:30  → 「這週有 3 件事還沒完成，要我幫你重新排程嗎？」
```

**3. 你說話，它整理 — 不需要任何格式**

不用學語法，不用選模板，不用點選單。  
說話，Kairo 理解你的意圖，自動建立正確類型的卡片。

**4. 完整閉環，不是工具，是系統**

```
你說話  →  捕捉識別  →  建立卡片  →  排程提醒  →  到時主動找你
   ↑                                                    ↓
你確認完成  ←  自動更新狀態  ←  跟進追蹤  ←────────────┘
```

**5. 數據永遠屬於你**

Notion 明天漲價或關閉，你的數據被綁架。  
Kairo 的所有記憶都是 Markdown 文件，在你自己的機器上，永遠可讀，永遠可遷移。

## 核心能力

### 🎯 智能捕捉 — 說話就夠了

> 「提醒我下週一早上9點跟 Jason 確認合同」  
> 「記一下：Naval 說的 Leverage 理論值得深入研究」  
> 「周五前要把報告發給王總」

支援 **10 種意圖類型**，置信度 ≥ 85% 自動建卡，低於時顯示選單確認：

| 類型        | 說明                   |
| ----------- | ---------------------- |
| `action`    | 需要執行的任務         |
| `timeline`  | 有 deadline 的事項     |
| `watch`     | 需要持續跟進的事       |
| `idea`      | 靈感 / 想法            |
| `memory`    | 要記住的人或事         |
| `belief`    | 認知 / 原則 / 思維框架 |
| `reference` | 資料 / 連結 / 論文     |
| `question`  | 待解答的問題           |
| `highlight` | 值得摘錄的金句         |
| `person`    | 聯絡人資訊             |

### 🗄 本地文件系統 — 你的數據，完全掌控

所有信息以 **Markdown 文件**形式存於本地，人類可讀，永不鎖定：

```
~/.openclaw/workspace/
│
├── 00_inbox/              ← 所有原始輸入，永不刪除
├── 02_work/
│   ├── tasks/             ← ⚡ 任務卡片（每個任務一個文件）
│   ├── tasks_master.md    ← 任務總索引
│   ├── waiting.md         ← 👀 跟進中清單（含 checkpoint）
│   ├── calendar.md        ← 📅 每日行程（Feishu Calendar 自動同步）
│   └── gmail.md           ← 📧 郵件摘要（每日 07:10 自動寫入）
├── 03_life/
│   └── daily_logs/        ← 📝 每日記憶（LLM 自動彙整）
└── 04_knowledge/
    ├── people/            ← 🧑 聯絡人卡片（對話中自動更新）
    ├── beliefs/           ← 💡 決策智慧（Naval/Munger/Dalio 等7位）
    └── roadmap.md         ← 長期路線圖
```

## 快速開始

> ⚠️ **這是 Ken 的個人生產環境版本。** 公開易用版 Kairo 正在規劃中。

### 環境要求

- Node.js >= 22 · pnpm >= 9 · Linux / macOS

### 安裝

```bash
git clone https://github.com/sou350121/Kairo-KenVersion.git
cd Kairo-KenVersion
pnpm install && pnpm ui:build && pnpm build
```

### 最小配置（3項）

```yaml
# ~/.openclaw/openclaw.json

channels:
  telegram:
    enabled: true
    token: "YOUR_BOT_TOKEN" # BotFather 申請

agents:
  defaults:
    model:
      primary: "openai-codex/gpt-5.3-codex" # 或任何 OpenAI-compatible model

gateway:
  port: 18789
```

### 啟動

```bash
node scripts/run-node.mjs gateway --port 18789
# 控制台：http://localhost:18789
```
